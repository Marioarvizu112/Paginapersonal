Proyecto Redes LAN Profesional

Uso de Redes LAN en una Organización

Las redes LAN (Local Area Network) permiten la interconexión de dispositivos dentro de un área limitada como una empresa, escuela u oficina. Su objetivo principal es facilitar la comunicación, compartir recursos (impresoras, archivos, internet) y mejorar la eficiencia operativa.

Ventajas:

  • Alta velocidad de transferencia de datos
  • Seguridad controlada dentro de la organización
  • Reducción de costos operativos
  • Facilidad de mantenimiento

Cableado Estructurado

El cableado estructurado es un sistema organizado de cables que permite la transmisión de datos de manera eficiente. Incluye cables UTP, fibra óptica, racks y paneles de parcheo.

Tipos de cables:

  • UTP Cat5e / Cat6
  • Fibra óptica
  • Cable coaxial

Equipos de Comunicación

Los dispositivos de red permiten la transmisión de datos entre equipos:

  • Switch: conecta múltiples dispositivos en la red
  • Router: conecta la red local a internet
  • Servidor: administra recursos y datos
  • Firewall: protege contra accesos no autorizados

Mantenimiento Preventivo y Correctivo

El mantenimiento es fundamental para el correcto funcionamiento de la red:

  • Preventivo: limpieza de equipos, revisión de cables, actualización de software
  • Correctivo: reparación de fallas, reemplazo de componentes dañados

Memoria Técnica

Documento donde se detalla la instalación de la red, incluyendo:

  • Planos de red
  • Direcciones IP
  • Configuración de dispositivos
  • Diagramas de conexión

Aspecto Legal y Normatividad

Delitos informáticos: acceso no autorizado, robo de información, sabotaje de sistemas.

Legislación de software: uso de licencias legales, evitar piratería.

Derechos de autor: protección del software y contenido digital.

Normativa: uso responsable del equipo de cómputo dentro de la organización.
